## üöÄ Getting Started

### Prerequisites
1. Visual Studio Code installed
2. GitHub Copilot extension installed and activated
3. Active GitHub Copilot subscription

### How to Use This Repository
1. Start with scenario 01 and progress sequentially
2. Read the comments and instructions in each file
3. Follow the exercises marked with `TODO:` or `EXERCISE:`
4. Experiment with your own variations

### Tips for Success
- **Be patient**: Copilot suggestions may take a moment to appear
- **Tab to accept**: Press Tab to accept a suggestion
- **Esc to dismiss**: Press Escape to dismiss unwanted suggestions
- **Alt+] / Alt+[**: Cycle through alternative suggestions
- **Ctrl+Enter**: Open Copilot completions panel for more options
